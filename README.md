# Getting Started
1.	Install dependencies
2.	Run project
3.  Build project


1. Install dependencies:

   `yarn install` or `npm install`

2. Run project:

   `yarn dev` or `npm dev`

   _(this command starts local dev server on http://127.0.0.1:5173/)_

3. Build project

   `yarn build` or `npm build`

   _(all files and assets will be compiled into **public** directory)._
